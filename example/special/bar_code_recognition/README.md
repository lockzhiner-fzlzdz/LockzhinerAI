## 1 简介

条码（Bar Code）是一种二维条形码，因其高效的编码和解码能力而得名。条码识别是视觉模块经常使用到的功能之一，经常用于识别超市的货物信息。本章节中，我们将教会你如何使用 Lockzhiner Vision Module 进行条码识别。
