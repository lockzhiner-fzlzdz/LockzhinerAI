## 1 简介

作为连接模拟世界与数字世界的桥梁，ADC 能够将连续的模拟信号（如温度、压力、声音等）转换为离散的数字信号，使得主控芯片能够处理和分析这些信号。本章节中，我们将教会你如何使用凌智视觉模块读取 ADC 数据。
